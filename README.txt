Problem Set 1
=============

The problem set itself can be found in `pset1_prompt.pdf`.

There are two parts to the problem set.

  - **Part A** expects written solutions.
  - **Part B** expects an implementation in python

## Submission Instructions

Please go to [alg.csail.mit.edu](http://alg.csail.mit.edu/) to submit your solutions. You should submit:

  - A **.pdf** file for your solutions to Part A. Please use `pset1_template.tex` to write your answers, then compile it to PDF.
  - Your `docdist.py` file containing your implementation for Part B.
